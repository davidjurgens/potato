# Potato LLM Preference Arena

Live demo of [Potato](https://github.com/davidjurgens/potato) for RLHF-style preference annotation.

Compare LLM responses side-by-side with:
- Pairwise scale rating (which response is better overall)
- Binary dimension judgments (helpfulness, accuracy)
- Free-text justification

## Try It Out

1. Enter any username to log in (no password required)
2. Read the prompt and both responses (A and B)
3. Rate your overall preference on the scale
4. Judge helpfulness and accuracy
5. Click Next to proceed

## About Potato

Potato is a free, open-source annotation platform supporting 20+ annotation types. [Learn more on GitHub](https://github.com/davidjurgens/potato).
