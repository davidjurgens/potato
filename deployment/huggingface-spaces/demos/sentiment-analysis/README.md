# Potato Sentiment Analysis Workbench

Live demo of [Potato](https://github.com/davidjurgens/potato), the portable annotation tool for NLP research.

This Space showcases **sentiment analysis annotation** with:
- Radio button classification (positive / negative / mixed / neutral)
- Likert confidence scale
- Span-level sentiment phrase marking
- AI-assisted keyword highlighting

## Try It Out

1. Enter any username to log in (no password required)
2. Read the product/service review
3. Classify the overall sentiment and rate your confidence
4. Optionally highlight sentiment-bearing phrases
5. Click Next to proceed

## About Potato

Potato is a free, open-source annotation platform supporting 20+ annotation types, AI-assisted annotation, quality control, and more. [Learn more on GitHub](https://github.com/davidjurgens/potato).
